Fragile™ Help Page

Fragile™ is a user friendly calculator developed by Sagacious Media.

Main Window

The following is an example of the main window.

The user has the ability to either enter values using the physical keyboard, or by using the soft keyboard on the calculator.

Current Mixed Fraction

The current mixed fraction is the fraction at the bottom right of the display. This fraction is the one the user is currently inputting. It goes to the top left after an operator or the equals has been clicked.

Current Expression

The expression in the top left, made up of a left operand, a operator, and a right operand, shows the whole expression including the evaluated expression.

Running Calculations

The user has the choice to use the previous evaluated expression as one of the operands in the next expression, or to start over.

Buttons

Reset

When the reset button is hit, the current expression and the current operand are cleared.

Clear

When the clear button is hit, the current mixed fraction is cleared.

Backspace

When the backspace button is clicked, the most recent input is deleted.

Toggle Sign

When clicked, the sign of the current mixed fraction changes.

Position (P)

Changes the focus on the current mixed fraction.

1-9

Inputs the respective number into the focused portion of the current mixed fraction.

Operators:

Addition Subtraction Division Multiplication Inverse

Each of these operators do their expected calculation.

Simplify

Simplies the current mixed fraction.

Mediant

Calculates the mediant.

Integer Power

Calculates the integer power of a given mixed fraction.

Relational Operator

Calculates the relational operation between 2 given mixed fractions, returning a boolean in a pop up window.

Menu Bar

File->Exit

Closes and terminates the calculator.

File->Print Session

Prints the current session.

File->New Calculator

Opens a new calculator with no saved preferences

File->Open Recording

Opens a place for the user to enter a file with a recording.

File->Save Recording

Opens a place for the user to enter a file to save the recording to.

Preferences->Open | Preferences->Edit | Preferences->Save

Opens the current prefernces that have been saved, allows you to edit them, as well as save inputted preferences to be used in other sessions.

Mode->Proper | Mode->Reduced

Allows the user to choose how the fractions are displayed by selecting a mode.

View->Pie Chart

Provides a pie chart representation of the current expression.

Style->Bar | Style->Slash | Style->Solidus

Allows the user to choose the style of the fraction bar.

Help->About

Opens up an about page.

History

The history can be seen by the user via a pull out tab that must be clicked. The history stores previous expressions, and allows the user to copy and paste previous expression.

The user then clicks the '>'

Recording Controls

The controller has three buttons, the stop, pause, and record. Once the user clicks record, the calculator will save the inputs and outputs. If the user hits pause, the recording pauses. If the user hits stop, the recording is stopped and saved.

Playback Controls

The controller has 3 buttons and a slider. The slider controls the speed of the playing. The stop button stops the system and closes it. Once the start button is clicked, if the pause button is clicked it is paused until the user clicks the play button again.

Intermediate Steps

The intermediate steps can be seen by the user via a pull out tab that must be clicked. The intermediate tab stores the explanation of the most recent expression in steps.

The user then enters an expression and clicks the '>'
